## Modifying Generated HTML Output

The HTML output is generated by `generate_html()` in **app.py**.

### Important: Update Both Files

If you change HTML generation, you must update **both**:
- `app.py` - for the web application
- `generator.py` - for the CLI tool

### Structure of generate_html()

```python
def generate_html(data):
    title = data.get('title', 'Cheatsheet')
    sections = data.get('sections', [])

    # 1. CSS styles (embedded in <style>)
    css = '''
        body { font-family: ... }
        .section { ... }
        @media print { ... }
    '''

    # 2. Build HTML body
    body_html = ''
    for i, section in enumerate(sections):
        body_html += f'<div class="section">...'

    # 3. JavaScript (embedded in <script>)
    js = '''
        function toggleSection(el) { ... }
        function printCheatsheet() { ... }
    '''

    # 4. Combine everything
    return f'''<!DOCTYPE html>
<html>
<head><style>{css}</style></head>
<body>{body_html}<script>{js}</script></body>
</html>'''
```

### Example: Adding a Footer

To add a footer to generated cheatsheets:

```python
# In generate_html(), before the closing </body>

footer_html = f'''
<footer class="cheatsheet-footer">
    Generated on {datetime.now().strftime('%Y-%m-%d')}
</footer>
'''

# Add CSS for the footer
css += '''
.cheatsheet-footer {
    text-align: center;
    padding: 20px;
    color: #666;
    font-size: 12px;
}
@media print {
    .cheatsheet-footer { display: none; }
}
'''
```

## Modifying the Editor UI

The editor UI consists of three files:

| File | What to Modify |
|------|----------------|
| `templates/index.html` | HTML structure, layout |
| `static/js/app.js` | Behavior, interactivity |
| `static/css/style.css` | Visual styling |

### Adding a New Button to the Toolbar

1. **Add HTML** in `templates/index.html`:
```html
<div class="toolbar">
    <!-- Existing buttons -->
    <button id="my-new-button" title="My Feature">
        My Feature
    </button>
</div>
```

2. **Add JavaScript** in `static/js/app.js`:
```javascript
document.getElementById('my-new-button').addEventListener('click', function() {
    // Your functionality here
    console.log('Button clicked!');
});
```

3. **Add CSS** in `static/css/style.css` (if needed):
```css
#my-new-button {
    background: #4CAF50;
    color: white;
}
```

### Adding a New Input Field to Sections

1. **Add to renderSections()** in app.js:
```javascript
function renderSections() {
    // Inside the section template
    html += `
        <div class="section-field">
            <label>New Field:</label>
            <input type="text"
                   value="${section.newField || ''}"
                   onchange="updateSection(${i}, 'newField', this.value)">
        </div>
    `;
}
```

2. **Add handler function**:
```javascript
function updateSection(index, field, value) {
    saveState();
    state.sections[index][field] = value;
    // Don't re-render to avoid losing focus
}
```

## Adding API Endpoints

### Basic Route Pattern

```python
@app.route('/api/your-endpoint', methods=['GET', 'POST'])
def your_endpoint():
    if request.method == 'POST':
        data = request.json
        # Process data
        return jsonify({'success': True})
    else:
        # Return data
        return jsonify({'data': 'value'})
```

### Example: Export Stats Endpoint

```python
@app.route('/api/stats')
def get_stats():
    """Get statistics about all cheatsheets."""
    cheatsheets = list_cheatsheets()

    total_sections = 0
    total_lines = 0

    for cs in cheatsheets:
        data = load_cheatsheet(cs['id'])
        if data:
            for section in data.get('sections', []):
                total_sections += 1
                total_lines += len(section.get('lines', []))

    return jsonify({
        'total_cheatsheets': len(cheatsheets),
        'total_sections': total_sections,
        'total_lines': total_lines
    })
```

## Modifying Print Layout

### Print CSS Location

In `generate_html()`, find the `@media print` block:

```python
css = '''
@media print {
    /* Hide UI elements */
    .header { display: none; }
    .index-header { display: none; }

    /* Layout */
    .sections {
        display: grid;
        grid-template-columns: 1fr 1fr;  /* 2 columns */
    }

    /* Prevent breaking subsections */
    .subsection {
        break-inside: avoid;
    }

    /* Page margins */
    @page {
        margin: 1cm;
    }
}
'''
```

### Common Print Modifications

**Change number of columns:**
```css
.sections {
    grid-template-columns: 1fr 1fr 1fr;  /* 3 columns */
}
```

**Add page breaks:**
```css
.section {
    break-before: page;  /* Each section on new page */
}
```

**Change margins:**
```css
@page {
    margin: 0.5in;  /* Smaller margins */
}
```

### Print JavaScript

The `printCheatsheet()` function ensures all content is visible:

```javascript
function printCheatsheet() {
    // Force content visible
    document.querySelectorAll('.section-content, .subsection-content')
        .forEach(el => el.style.display = 'block');

    setTimeout(() => {
        window.print();

        // Restore after print
        setTimeout(() => {
            document.querySelectorAll('.section-content, .subsection-content')
                .forEach(el => el.style.display = '');
        }, 500);
    }, 100);
}
```
